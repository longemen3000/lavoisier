UML Model

 

You can browse the UML Model of CAPE-OPEN SMST Component. It contains four packages :

  • Use-Case View : use-cases model, sequence and collaboration diagrams.

  • Logical View : interface and state diagrams.

  • Component View : packages and components diagrams.

  • Deployment View : deployment diagram.

 

 

Open the model using a visual modelling tools :

  • Rose format - You just need the Rational Rose 98 (or upper) software !


Notes :
The model was designed with Rational Rose 98 SP1 - Rose Professional Edition - 4.5.8163.3
If possible, give your preference to this native file. You have all surfing capabilities.

 

Open the model using alternative formats :

  • HTML - Go on with your browser.

    • HTML format n°1
      Nice because it keeps the organization of Rose file. Unfortunately, this format doesn't display the Component View, the Deployment View and the packages descriptions (it's a pity).

    • HTML format n°2
      All information but a little bit confused (doesn't keep the organization of Rose file)
      Each view has a link : Use-Case View, Logical View, Component View, Deployment View.

  • Microsoft Word 97 - For printing capabilities even if some figures are too large.
    Each view has a link : Use-Case View, Logical View, Component View, Deployment View.


Notes :
You should open the HTML formats using a new window in order to come back here quickly.
You could have some problems with the banner of the HTML formats according to your browser.
