CORBA's IDL

 

 

These 3 files contains the IDL definition of the CO SMST Component. Compile them using your IDL compiler and create stub and skeleton.

 

ICapeNumericSMST.idl : Interface Definition for SMST

ICapeUtilityDefinition.idl : Types and Exceptions Definition

ICapeUtility.idl : Interface Definition for CAPE-OPEN Common services

 


Notes :

These CORBA's IDL links displays the defining lines with some very basic comments.
You should refer to the CAPE-OPEN Common Services inside the CAPE-OPEN Concepts Document for the real content of ICapeUtility.idl.
The compilation should be successfull from idl compiler compliant with CORBA 2.0 and upper.
