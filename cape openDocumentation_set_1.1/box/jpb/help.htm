Which tools to view this documentation ?

This documentation has been tested sucessfully with these browsers :

  • Microsoft Internet Explorer 4.01 and upper,

  • Netscape Navigator 4.00 and upper.

 

Which operating system to view this documentation ?

This documentation has been tested sucessfully on Win32 (Windows 9x, NT) and Digital Unix System v4.0.

 

Do we need to have another software than a browser ?

No, but the UML model has been designed using the Rational Rose 98 software. So you need to have Rational Rose 98 or upper (standard, professional or enterprise edition) to view directly this native format.
Otherwise, this model is also proposed in full HTML. Therefore, you can go on with your browser. It is also available for Microsoft Word 97.

 

Are there some tips to view the documentation ?

  • No page has link to the table of contents page except this one (too laborious to do that for each page !). So use the back function of your browser or more efficiently save the table of contents link.

  • If you own Rational Rose 98 or upper (standard, professional or enterprise edition), download the native file of the UML model and open it rather than using the HTML or MS Word version.
