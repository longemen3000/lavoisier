Flowsheet Analyser 1.0

 

 

A complete Java application called Flowsheet Analyser has been developed for the validation step of CAPE-OPEN project. The goal is to show how the client side of the SMST Interface may be implemented. The Flowsheet Analyser uses services coming from any SMST component compliant with CAPE-OPEN version 2.0 Interface Specification (for instance the Flowsheet Server 5.0 application).

The Flowsheet Analyser is designed to model flowsheet, as well as to to perform analysis of a number of flowsheets. An easy to use graphical user interface is provided. It is designed with respect to the Command pattern.
Through the GUI, you can load and build the flowsheets described in the Set of Client Programs part. Then, you can view the selected flowsheet (this description is only textual) and/or perform the numerical analysis you select (bind to the Flowsheet Server). The result of computation is displayed within a textual area.


 

This application is deployed, you can download and install it from the Appendices.
