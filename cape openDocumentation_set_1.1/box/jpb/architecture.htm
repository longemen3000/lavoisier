Architecture

 

 

The architecture of the CO system is based on an object-oriented technology, which allows software systems to be constructed from software components. These components are able to talk to each other via defined interfaces. The software components can come from different vendors and may reside on the same machine or be on different machines across a network. The CAPE-OPEN Interface System is a standard means of connecting an external software component to any compliant simulator. The Interface can be "socket" and "plug", which exchanges information between the two parts. The simulator and the SMST component do not have to know anything about the internal coding and standards used by the other. The job of the interface is to translate requests for information or action by either party into something the other understands.
