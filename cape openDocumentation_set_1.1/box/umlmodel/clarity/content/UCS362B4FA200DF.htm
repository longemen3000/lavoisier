Add Stream

Add Stream Use-Case.

Description:
In order to build a Process Graph, all streams of the flowsheet must be described by giving the initial Unit each stream comes from, as well as the final Unit each stream goes to.
If these units do not exist, they are created. If the unit(s) already exist(s), that is implementation dependent.
The stream has by default a weight (implementation dependent) and properties (To Be Torn = False and Not To Be Torn = False). Use [Add Weight] and [Add Stream Property] to overwrite them.
