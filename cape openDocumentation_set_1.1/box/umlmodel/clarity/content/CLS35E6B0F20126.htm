Class: <<Interface>> ICapeNumericSequencing

Description:

The ICapeNumericSequencing Interface.

Use this to perform a Sequencing Analysis on a Flowsheet.
The flowsheet do not have any recycle.

Superclasses:

<<Interface>>ICapeNumericAnalysis The ICapeNumericAnalysis Interface.

This is the basic Interface of numerical Analysis model: the minimum requirements for an analysis.

It is an "abstract" Interface. It is not distributed. So, no client can get its object reference.
