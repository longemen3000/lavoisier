Get Initial Graph

Get Initial Process Graph Use-Case.

Description:
Gets the "parent" graph from which the graph has been created.
A Graph B can be generated from the analysis results of a Graph A. The Graph B keeps in mind the "parent" Graph A which has created it. This Use-Case gives this "parent" Graph. If this "parent" Graph does not exist, a null value is returned.

Assumptions:
None.

Exceptions:
None.
