Check Available Analysis

Check Available Analysis Use-Case.

Description:
Checks if a numerical analysis is available. The result is TRUE if a numerical analysis available, otherwise the result is FALSE.
The numerical analysis are PARTITIONING, TEARING, SEQUENCING and (FULL) SEQUENTIAL MODULAR ANALYSIS.
If it is not available (for instance, it is not implemented), the corresponding computation is not possible.

Assumptions:
None.

Exceptions:
None.
