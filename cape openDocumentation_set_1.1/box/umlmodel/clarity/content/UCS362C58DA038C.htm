Create a Partition Graph from Partitioning Analysis Result

Create a Partition Graph from Partitioning Analysis Result Use-Case.

Description:
According to the computation result, from the flowsheet creates and generates a Partition Graph for a specified partition.
The intermediate streams between partitions become FEED or PRODUCT streams of the built Partition Graph. These streams use the External System Representation through [Get External System Representation].

Assumptions:
None.

Exceptions:
None.
