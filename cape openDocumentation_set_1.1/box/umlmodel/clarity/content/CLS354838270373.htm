Class: <<Interface>> ICapeNumericProcessGraph

Description:

The ICapeNumericProcessGraph Interface.

Use this to describe a Process Graph which is any Flowsheet.

Superclasses:

<<Interface>>ICapeNumericFlowsheet The ICapeNumericFlowsheet Interface.

This is the basic Interface of Flowsheet description model. Would allow to develop a more complex Flowsheet description model.

It is an "abstract" Interface. It is not distributed. So, no client can get its object reference.

Subclasses:

<<Interface>>ICapeNumericPartitionGraph The ICapeNumericPartitionGraph interface.

Use this to describe a Partition Graph which is a Flowsheet having only one partition.
