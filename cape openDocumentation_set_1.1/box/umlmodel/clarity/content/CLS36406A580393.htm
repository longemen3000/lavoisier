Class: <<Interface>> ICapeNumericAnalysis

Description:

The ICapeNumericAnalysis Interface.

This is the basic Interface of numerical Analysis model: the minimum requirements for an analysis.

It is an "abstract" Interface. It is not distributed. So, no client can get its object reference.

Superclasses:

<<CO Common Service>>Parameter Set of Interfaces including the Parameter Concept.

It specifies the CAPE-OPEN common services on the Parameter concept.

No information on its content is given here, please refer to the CAPE-OPEN Concepts Document.

Subclasses:

<<Interface>>ICapeNumericTearing The ICapeNumericTearing Interface.

Use this to perform a Tearing Analysis on a Flowsheet.
<<Interface>>ICapeNumericSMAnalysis The ICapeNumericSMAnalysis Interface.

Use this to perform a Full Sequential Modular Analysis on a Flowsheet.
<<Interface>>ICapeNumericPartitioning The ICapeNumericPartitioning Interface.

Use this to perform a Partitioning Analysis on a Flowsheet.
<<Interface>>ICapeNumericSequencing The ICapeNumericSequencing Interface.

Use this to perform a Sequencing Analysis on a Flowsheet.
The flowsheet do not have any recycle.
