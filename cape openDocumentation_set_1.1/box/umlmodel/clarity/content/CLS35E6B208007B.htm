Class: <<CO Common Service>> Parameter

Description:

Set of Interfaces including the Parameter Concept.

It specifies the CAPE-OPEN common services on the Parameter concept.

No information on its content is given here, please refer to the CAPE-OPEN Concepts Document.

Subclasses:

<<Interface>>ICapeNumericFlowsheetFactory The ICapeNumericFlowsheetFactory Interface.

This is the factory of a Flowsheet model.
<<Interface>>ICapeNumericAnalysis The ICapeNumericAnalysis Interface.

This is the basic Interface of numerical Analysis model: the minimum requirements for an analysis.

It is an "abstract" Interface. It is not distributed. So, no client can get its object reference.
<<Interface>>ICapeNumericFlowsheet The ICapeNumericFlowsheet Interface.

This is the basic Interface of Flowsheet description model. Would allow to develop a more complex Flowsheet description model.

It is an "abstract" Interface. It is not distributed. So, no client can get its object reference.
<<Interface>>ICapeNumericGATComponent The ICapeNumericGATComponent Interface.

This is the entrance object to the CO SMST Component.
First of all, the client must bind to this interface or get its reference by another way. Then, the client can access to the CO SMST Component.
<<Interface>>ICapeNumericAnalysisFactory The ICapeNumericAnalysisFactory Interface.

This is the factory of a numerical Analysis.
